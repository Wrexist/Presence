Features

🗺️
Ambient Presence Map P0
Full-screen MapKit with Liquid Glass overlays. Glowing aurora dots show who's Present nearby. Max 50 dots, clustering beyond. PostGIS radius queries.
✨
Presence Toggle P0
Glass pill "Go Present" button. Opts in, starts location (whenInUse, ±50m reduced precision). Auto-expires in 3 hours.
👋
Wave System P0
Tap a dot → glass sheet → AI icebreaker loads → wave → mutual wave unlocks 10-minute timed chat → connection recorded.
🤖
AI Icebreaker (Opus 4.7) P0
Claude generates a single contextual opener based on venue type, time, and both users' 3-word bios. Max 200 chars. ~800ms. Fallback library for outages.
⏱️
10-Minute Chat Window P0
Sacred constraint. Countdown chip visible. Chat locks at 00:00 with "Head over and say hi!" No history stored. Luma celebrates after.
💰
Presence+ Subscription P1
3 presences/week free. Presence+ = unlimited + see nearby count before activating + Luma skins. $6.99/mo or $49.99/yr via RevenueCat.

Luma — The Mascot

The Research Case
Why Presence needs a mascot — data-backed

20–40%
Higher user retention for apps with mascots vs. without
+5%
DAU lift from adding mascot to push notifications
∞
Fan art, memes, organic brand amplification. Discord's Wumpus has more fan art than most indie games.

The loneliness app category has a unique stigma problem — users feel vulnerable using it. A mascot transforms the emotional frame: from "I'm using an app because I'm lonely" (shame) → "I'm connecting with Luma's world" (delight). The mascot creates a third party in the relationship that feels supportive, not transactional.

Luma's Identity
A bioluminescent deep-sea creature

In the deep ocean, creatures evolved to make their own light — not just to see, but to signal: I'm here. I'm safe. Come closer. Luma found the surface world and discovered something strange: all these beings, surrounded by each other, somehow alone in the dark. So Luma does what comes naturally — glows.

✦Non-gendered — universally inclusive
✦Bioluminescent — presence = light in darkness (on-brand)
✦Rounded, no sharp edges — maximum approachability
✦Small, slight — Luma supports, never dominates. User is the hero.
✦Never guilt-trips — anti-Duolingo aggressive notification pattern

iOS 26 · Liquid Glass

Apple's New Design Language — Fully Native
Presence is built iOS 26 first. Every navigation element, button, and overlay uses native Liquid Glass APIs — not custom blur.

// GlassTokens.swift — single source of truth enum GlassTokens { enum Radius { static let card: CGFloat = 28 static let pill: CGFloat = 999 static let sheet: CGFloat = 34 } } // Primary CTA — the "Go Present" button GlassPillButton("Go Present") { presenceService.activate() } .glassEffect(.regular, in: Capsule()) // Always gate for iOS < 26 fallback if #available(iOS 26.0, *) { view.glassEffect() } else { view.background(.regularMaterial) }
Liquid Glass Rules — What Presence Follows
✓ USES GLASS
→ Tab bar & navigation bar → All CTA buttons (Go Present, Wave) → Bottom sheets & modals → Venue info overlays → Status chips (timer, count) → Map control buttons
✗ NO GLASS
→ Map tiles (content layer) → Presence dot markers (custom glow) → Text content areas → List/scroll content → Stacked on other glass → Full-screen backgrounds
